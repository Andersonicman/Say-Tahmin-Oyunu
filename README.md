# 🎯 Sayı Tahmin Oyunu (Python)

Bu proje, Python öğrenme sürecimde geliştirdiğim basit bir sayı tahmin oyunudur.

## 📌 Proje Hakkında

Program, 1 ile 100 arasında rastgele bir sayı tutar ve kullanıcıdan bu sayıyı tahmin etmesini ister.
Kullanıcı doğru tahmini bulana kadar oyun devam eder.

## 🚀 Nasıl Çalışır?

* Bilgisayar 1-100 arasında rastgele bir sayı seçer
* Kullanıcıdan tahmin alınır
* Girilen tahmine göre yönlendirme yapılır:

  * Daha büyük sayı gir
  * Daha küçük sayı gir
* Doğru tahmin edildiğinde oyun biter
* Kaç denemede bulunulduğu ekrana yazdırılır

## 🛠️ Kullanılan Yapılar

* `random` modülü
* `while` döngüsü
* `if-elif-else` koşulları
* `input()` ve `print()` fonksiyonları

## ▶️ Nasıl Çalıştırılır?

1. Python yüklü olmalıdır
2. Dosyayı çalıştır:

```
python dosya_adi.py
```
